L-Band Digitally Tunable Filter

L Band Digitally Tunable Filter is a Varactor based tunable band pass filter High Q cavity based design. Precision machined for preserving Q and repeatability Digital or Analog Control for frequency Tuning.

Features

  • Frequency tuning via SPI lines
  • Input Power handling capability of 1 W
  • Fast settling time of 40 uSec
  • Automatic level correction facility by lookup table
  • Smaller dimensions
  • Operates on 28 V DC nominal voltage

Specifications

Frequency range 960 to 1240 MHz
Operating Power 3 dBm
Input Power Handling +30 dBm
Impedance 50 Ohms
3 dB Bandwidth 5%
Settling time 40 u Sec
30 dB/ 3 dB shape factor 7(typ)
Frequency Tuning SPI
ALC ALC by Look up table
Supply Input 18-32 V DC +/- 10%
Dimensions 80 x 60 x 22 mm
Operating Temperature -40°C to +71°C
