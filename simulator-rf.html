Simulator For RF module

Simulator For RF module of SDR is used for evaluation of RF Module. It simulates test conditions required for assuring proper functionality of the RF Module. It generates RF frequency between 30 MHz to 512 MHz for Rx test. It also generates IF frequencies of 21.4 MHz / 70 MHz as required for feeding to transmitter IF of RF Module. Simulator has provision for connecting passive load of 100W rating with 50 ohms impedance. Simulator has a built in controller to perform BITE on RF module. Front panel has display for configuration control and status indications.

Simulator For RF Module Features

Features

  • GUI based application Software
  • Micro-controller based circuit
  • USB/ RS232 interface circuits for remote operation
  • ADC and DAC for handling analog signals
  • Keypad for selecting frequency, power level, channel bandwidth
  • CW and pulse signal source
  • BITE facility
  • 50 Ohms, 100W CW passive load for terminating antenna port of RF Module
