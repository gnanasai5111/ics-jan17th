Receiver Exiter System

Receiver & Exciter system operates in three mutually exclusive modes; Pulsed mode, FMCW mode and Calibration mode. In pulsed mode exciter section generates appropriate power & frequency at RF Pulse O/P Port. The PRF signals input to the receiver & exciter system is used for synchronization & pulsed mode operation. In FMCW mode exciter section generates frequency modulated continuous wave signal with bandwidth of 200 MHz . In calibration mode exciter section generates pilot signal at calibration output port with appropriate frequency & power for calibration, for amplitude and phase of receiver channels. The RF Pulse O/P and FMCW O/P will be in the isolation/cut off mode.

Receiver Exiter System Features

Features

  • Pulsed mode
  • FMCW mode
  • Calibration mode
  • PRF signals for synchronization & pulsed mode operation
  • Homodyne configuration in FMCW mode
  • Pilot signal at calibration output in calibration mode
