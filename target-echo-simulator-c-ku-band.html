Target Echo Simulator - C and Ku Band

Radar Target Echo Simulator (RTES) is designed to test and evaluate functionality of various radar parameters. This is deployed in lab environment for simulating the target characteristics. C-Band and Ku Band TES is designed with digital signal processing and Up Down converter modules integrated with a user friendly software. Target Echo Simulator generates target echoes in phase coherent mode. RTES is fed with RF modulated with waveform like CW, Pulsed CW, LFM, Poly phase coded RF pulse, Non Linear FM. Commands defining RF frequency, frequency sequence in case of Frequency agility are fed on ethernet or RS 422.

Target Echo Simulator - C and Ku Band Features

Features

  • DRFM based architecture. DRFM methods are used in Radar jamming and deception applications
  • Digital module operates at intermediate frequency
  • Capturing an RF signal in digital format and retransmitting it after delay
  • Security for log in and user identification
  • User selectable features like center frequency, waveform type
  • Injection mode and Radiation Mode of operation
