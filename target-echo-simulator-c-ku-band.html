Target Echo Simulator - C and Ku Band

Radar Target Echo Simulator (RTES) is designed to test and evaluate functionality of various Radar parameters. This is deployed in Lab Environment by simulating the target characteristics. C-Band and Ku Band TES is designed with Digital Signal processing and Up down Converter modules integrated with a user friendly software. Target Echo Simulator generates target echoes in phase coherent mode. RTES is fed with RF modulated with waveform like CW, Pulsed CW, LFM, Poly phase coded RF pulse, Non Linear FM. Commands defining RF center frequency, frequency sequence in case of Frequency Agility are fed on Ethernet or RS 422

Features

  • DRFM based architecture. DRFM methods are used in Radar Jamming and Deception applications
  • Digital module operates at Intermediate Frequency
  • Capturing an RF signal in digital format and retransmitting it after delay
  • Security for Log IN and User Identification
  • User selectable features like Center Frequency, Waveform type
  • Injection mode and Radiation Mode of operation

Application

  • Radar Target Simulation, Test and Evaluation

Specifications

Input Frequency Range C Band and Ku Band
Input RF Bandwidth 200 MHz
Input Waveform Modulation CW, Pulsed CW, LFM, Poly phase coded RF pulse, Non Linear FM
Pulse Width Range 1 us (Min), 200 us (Max)
Input Sensitivity -50 dBm
Input Dynamic Range 50 dB
Output RF Power Range -90 dBm to +10 dBm
Pulse Repetition Interval 100 us (min), 1 ms (max)
Target Range 500 m to 150 km
Resolution 15 mt
Target Speed 1000 m/sec
Number of Targets 8 Targets
Noise Jamming Spot, Barrage
